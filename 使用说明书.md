# 爬虫小工具 - 使用说明书

## 一、功能概述

本工具用于采集抖音、快手、小红书、微信视频号的同城频道数据，支持语义分析、关注提醒、数据导出等功能。

## 二、环境要求

- Windows PC
- Python 3.10+
- 互联网连接

## 三、安装与启动

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 初始化爬虫（如需 Playwright）

```bash
playwright install chromium
```

### 3. 配置文件

将 `config.example.yaml` 复制为 `config.yaml`，按需填写：

- **llm**：大模型 API（硅基流动），用于语义判断
- **dingtalk**：钉钉机器人 Webhook
- **wechat**：企业微信机器人 Webhook（可推送至微信）

### 4. 启动程序

```bash
python main.py
```

## 四、登录

- 默认管理员：`admin` / `admin123`（首次登录后建议修改密码）
- 仅管理员可添加普通用户
- 普通用户无法修改/删除数据库

## 五、必做功能 - 数据采集

1. 在「采集平台」中勾选需要爬取的平台（可全选或单选）
2. 点击「开始采集」
3. 各平台并行采集，互不干扰
4. 采集速度以稳定为主，约 60 秒一轮
5. 数据自动去重后存入采集数据库
6. 点击「采集数据」选项卡中的「刷新」查看数据

**注意**：抖音、快手、小红书、微信视频号的实际爬取需根据各平台当前接口实现，当前为框架占位。如需真实数据，需接入官方接口、第三方数据服务或自行实现爬虫逻辑。

## 六、可选功能 - 语义判断

1. 在 `config.yaml` 中配置大模型 API Key 和模型
2. 点击「语义判断（批量送大模型）」
3. 系统将采集库中的数据分批发送至大模型
4. 大模型返回正面/中性/负面判断
5. 负面言论自动存入「负面言论数据库」
6. 可在「负面言论」选项卡查看、导出

## 七、可选功能 - 信息提醒

1. 添加关注对象：
   - 手动：点击「添加关注」，选择平台、填写 ID、昵称
   - 批量：使用 Excel 模板，包含「平台」「ID」「昵称」列，点击「Excel 导入关注」
2. 采集过程中，当关注对象有新发布时：
   - 弹出提示框
   - 播放提示音
   - 自动存入「关注对象数据库」
3. 可配置钉钉/企业微信，将新内容推送至手机

## 八、管理功能

- **导出**：采集数据、负面言论、关注对象均可导出为 Excel
- **备份**：备份数据库至 `data/backups/`
- **用户管理**（仅管理员）：添加用户、修改密码
- **清空采集库**（仅管理员）：慎用

## 九、数据存储

- 所有数据存储在本地 `data/` 目录
- 主数据库：`data/crawler.db`

## 十、常见问题

### 1. 采集无数据？

当前各平台爬虫为占位实现，需根据实际接口补充爬取逻辑。建议参考各平台开发者文档或使用合规数据接口。

### 2. 语义判断失败？

检查 `config.yaml` 中 API Key 和模型是否配置正确，硅基流动平台地址为 `https://api.siliconflow.cn/v1`。

### 3. 钉钉/微信推送失败？

确认 Webhook URL 正确，钉钉需创建群机器人，企业微信需创建应用机器人。
