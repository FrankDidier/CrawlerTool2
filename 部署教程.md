# 爬虫小工具 - 部署教程

## 一、环境准备

### 1. 安装 Python

- 下载 Python 3.10 或更高版本：https://www.python.org/downloads/
- 安装时勾选「Add Python to PATH」
- 验证：打开 cmd，输入 `python --version`

### 2. 安装依赖

在项目目录打开命令行：

```bash
cd 项目目录
pip install -r requirements.txt
```

若速度较慢，可使用国内镜像：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 3. 安装 Playwright（可选，用于浏览器爬虫）

若后续接入真实爬虫逻辑并采用 Playwright：

```bash
playwright install chromium
```

## 二、配置文件

1. 将 `config.example.yaml` 复制为 `config.yaml`
2. 按需填写：

```yaml
app:
  name: 爬虫小工具
  data_dir: ./data

llm:
  base_url: https://api.siliconflow.cn/v1
  api_key: "你的硅基流动API Key"
  model: "Qwen/Qwen2.5-7B-Instruct"

dingtalk:
  webhook_url: "钉钉群机器人Webhook"

wechat:
  webhook_url: "企业微信机器人Webhook"
```

## 三、首次运行

```bash
python main.py
```

首次运行将自动创建：

- `data/` 数据目录
- `data/crawler.db` 数据库
- 默认管理员账户：`admin` / `admin123`

**请登录后尽快修改管理员密码。**

## 四、生成关注对象导入模版

```bash
python create_watch_template.py
```

将生成 `关注对象导入模版.xlsx`，填写后可在程序中「Excel 导入关注」使用。

## 五、打包为可执行文件（可选）

使用 PyInstaller 打包为 exe，便于在无 Python 环境电脑运行：

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "爬虫小工具" main.py
```

生成的可执行文件在 `dist/爬虫小工具.exe`。

**注意**：打包后需将 `config.example.yaml`、`关注对象导入模版.xlsx` 等与 exe 放同一目录，或修改代码使配置路径正确。

## 六、Android 模拟器采集（高级，可选）

当抖音网页端因验证码无法搜索同城内容时，可使用 Android 模拟器方案。
工具会自动尝试此方案（方案5），若未配置则自动跳过。

### 1. 安装 Android SDK

- 下载 Android Studio：https://developer.android.com/studio
- 安装后打开 SDK Manager，安装 Android API 30+
- 创建一个 AVD（Android Virtual Device），推荐 Pixel 4, API 33

### 2. 安装 Appium

```bash
npm install -g appium
appium driver install uiautomator2
```

### 3. 安装 Python 客户端

```bash
pip install Appium-Python-Client
```

### 4. 在模拟器中安装抖音

- 启动模拟器: `emulator -avd <你的AVD名称>`
- 下载抖音 APK 并安装到模拟器: `adb install douyin.apk`
- 在模拟器中打开抖音，完成登录

### 5. 启动 Appium 服务

```bash
appium
```

### 6. 使用

配置好后，在工具的「设置 → 目标城市」填写城市名并开始采集。
当网页端方案 1-4 均失败时，工具会自动调用模拟器方案（方案5）。

## 七、常见部署问题

### 1. 提示找不到模块

确认已执行 `pip install -r requirements.txt`，且使用的 Python 版本正确。

### 2. 数据库被占用

关闭其他可能占用 `data/crawler.db` 的进程，或只运行一个程序实例。

### 3. 权限不足

以管理员身份运行，或将数据目录设置在用户有写权限的位置。
