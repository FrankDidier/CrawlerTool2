          同城数据采集 — 功能说明与使用教程
═══════════════════════════════════════════════════════════════

【背景】

抖音的「同城」标签页仅存在于手机 App 中，网页版无法直接访问
该频道。本工具通过调用 TikHub API 获取抖音「同城热点榜」数据，
这些内容是抖音根据城市维度推荐的热门视频，与 App 同城频道
展示的热门内容一致。

该功能通过 TikHub 服务实现（一个专业的社交媒体数据 API 平台），
用户只需在设置中填入 API Token 即可。

如未配置 API Token，工具会自动回退到城市话题搜索方案。

【试用模式】

本工具已内置一个试用 Token，首次使用可直接体验抖音同城采集。
试用 Token 的额度有限且为共享资源，仅供测试体验。
正式使用请注册自己的 TikHub 账号（免费）并替换为个人 Token。
在「设置」中可随时查看当前 Token 状态（试用 / 个人）。

───────────────────────────────────────────────────────────────
一、抖音采集策略（3 个方案 + 兜底）
───────────────────────────────────────────────────────────────

  ★ 方案0: 同城热点 API（推荐，优先执行）
  ├─ 原理：通过 TikHub API 调用抖音同城热点榜接口
  │        返回抖音按城市推荐的热门视频内容
  ├─ 前提：在「设置 → 抖音同城API」中填入 TikHub Token
  ├─ 优势：无需浏览器、无需登录、无验证码、真实城市内容
  ├─ 自动分页：每次采集自动翻页获取更多内容（默认3页）
  └─ 支持城市：全国 130+ 城市（自动匹配城市行政区划代码）

  方案1: 城市话题搜索（自动）
  ├─ 原理：在 douyin.com 搜索「{城市}同城」「{城市}生活」等话题
  ├─ 关键：使用已登录的 Cookie
  ├─ 模式：无头浏览器（不弹窗），全自动
  └─ 注意：当 API 方案不可用时的主要备选

  方案2: 城市话题搜索（手动验证）
  ├─ 原理：与方案1 相同，但使用可见浏览器
  ├─ 模式：弹出 Chrome 窗口，需手动完成验证码
  └─ Cookie 保存后后续轮次可复用

  兜底: 主页推荐流
  ├─ 当上述方案均未成功时自动触发
  └─ 注意：全国推荐，非同城；界面会提示「非同城，仅作样本」

───────────────────────────────────────────────────────────────
二、注册自己的 TikHub 账号（5分钟，免费）
───────────────────────────────────────────────────────────────

  ★ 为什么要注册？
    试用 Token 额度有限且共享，注册个人账号后：
    · 拥有独立额度，不受他人使用影响
    · 每日签到可持续获取免费额度
    · 需要时可自行充值，按需扩容

  第1步：注册 TikHub 账号
    访问 https://user.tikhub.io/users/signin
    点击页面下方「Sign up now」链接
    填写邮箱和密码完成注册

  第2步：验证邮箱
    注册后会收到一封验证邮件
    点击邮件中的链接完成验证
    ★ 如未收到，检查垃圾箱或在登录后重新发送

  第3步：每日签到领取免费额度
    登录后点击右上角「Check-in」签到按钮
    每天签到可获得免费 API 调用额度
    ★ 养成签到习惯即可长期免费使用！

  第4步：创建 API Token
    进入 API Key 管理页面：
    https://user.tikhub.io/users/api_keys
    点击「Create API Key」
    填写名称（如 "my-crawler"）
    勾选所有 Scopes 权限
    点击创建
    ★★★ 立即复制保存，Token 创建后只显示一次！★★★

  第5步：在工具中配置
    打开「设置」→ 找到「抖音同城API（★ 真实同城数据）」
    将 Token 粘贴到「API Token」输入框
    API 地址保持默认（海外用户：api.tikhub.io）
    ★ 国内用户请改为：api.tikhub.dev（无需VPN）

  第6步：测试连接
    点击「测试 API」按钮，确认显示"测试成功"

  第7步：开始采集
    回到主页，填写目标城市，点击「开始采集」
    状态栏会显示"★ 方案0: 调用真实同城API..."

───────────────────────────────────────────────────────────────

四、微信视频号采集策略（3 个方案）
───────────────────────────────────────────────────────────────

  方案1: 隐身模式
  ├─ 使用反检测补丁隐藏自动化特征
  └─ 全自动

  方案2: 持久化浏览器

  ├─ 使用默认浏览器连接
  └─ 全自动

五、使用步骤
───────────────────────────────────────────────────────────────

  1. 打开工具，登录（默认：admin / admin123）

  2. 进入「设置」：
     · ★「抖音同城API」— 填入 TikHub Token（强烈推荐！）
     ·「目标城市」— 填写城市名（如：深圳、北京、呼伦贝尔）
     · 各平台「登录」— 点击登录按钮完成登录
     ·「大模型配置」— 如需情感分析，填写 API 地址和 Key
     ·「机器人配置」— 如需告警推送，填写钉钉/企微 Webhook

  3. 回到「采集」页面：
     · 勾选要采集的平台
     · 点击「开始采集」

  4. 观察状态栏信息：
     · "═══ 第 1 轮采集开始 ═══"
     · "[抖音] ★ 方案0: 调用真实同城API..."
     · "[抖音API] 同城热点第 1 页（城市: 深圳, 代码: 440300）..."
     · "[抖音] ★ 同城API成功！获取 30 条城市热门内容"

  5. 采集完成后，可在「分析」页面进行情感分析，或「导出」数据

七、常见问题
───────────────────────────────────────────────────────────────

  Q: 采集到的内容和 App 同城频道一样吗？
  A: 如果配置了 TikHub Token，工具会获取抖音按城市维度推荐
     的热门内容，这些内容与 App 同城频道的热门推荐一致。

  Q: 不配置 API Token 能用吗？
  A: 可以。工具会自动回退到城市话题搜索方案，但效果不如
     API 方案。强烈建议配置 API Token。

  Q: TikHub 需要付费吗？
  A: 注册免费，每日签到可获取免费调用额度。
     日常使用完全够用。如需更多额度可购买。

  Q: 还需要每次都过验证码吗？
  A: 如果使用 API 方案（方案0），完全不需要任何验证码。
     API 方案不涉及浏览器，也不需要登录抖音。

  Q: 国内用户无法访问 api.tikhub.io？
  A: 国内用户请在「设置」中将 API 地址改为：
     https://api.tikhub.dev
     该域名可直接访问，无需代理。

  Q: 试用 Token 和个人 Token 有什么区别？
  A: 试用 Token 是共享的，额度有限，仅供体验测试。
     个人 Token 拥有独立额度，可每日签到获取，更稳定可靠。

  Q: 如何从试用切换到个人 Token？
  A: 在「设置」中将「API Token」替换为自己创建的 Token，
     点击「保存设置」即可。状态会从"试用 Token"变为"个人 Token"。

  Q: 如何切回试用 Token？
  A: 在「设置」中点击「恢复试用 Token」按钮即可。

  Q: Token 忘记复制了怎么办？
  A: 需要在 TikHub 删除旧 Token 并重新创建一个新的。

  Q: 我的城市不在列表中怎么办？
  A: 可联系技术支持添加城市代码。TikHub 的城市列表接口
     也可查询所有支持的城市代码。
